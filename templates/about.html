About This Project

This project is a Crop Predictor that uses Machine Learning to recommend the most suitable crop to grow based on key agricultural parameters.

It considers environmental and soil features like nitrogen (N), phosphorus (P), potassium (K), temperature, humidity, pH level, and rainfall to make accurate predictions.

The model is trained on a real-world agricultural dataset and helps farmers or users make informed decisions about what crop is best suited for their specific conditions â€” improving yield and efficiency.

Built using Flask, Python, and Bootstrap, this app provides a simple and intuitive interface where users can input field data and get real-time crop recommendations.
